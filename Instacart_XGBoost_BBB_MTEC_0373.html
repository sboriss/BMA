Determine product characteristics:

  • prod_orders: how many times a product_id was ordered by all users
  • prod_reorders: how many times a product_id was reordered by all users
  • prod_first_orders: number of times a product_id was ordered for the first time
  • prod_second_order: number of times a product_id was ordered for the second time
# Products ----------------------------------------------------------------
rm( prd_tmp, prd_tmp2 )

prd <- orders_products %>%
  arrange(user_id, order_number, product_id) %>%
  group_by(user_id, product_id) %>%
  mutate(product_time = row_number()) %>%
  ungroup() %>%
  group_by(product_id) %>%
  summarise(
    prod_orders        = n(),
    prod_reorders      = sum(reordered),
    prod_first_orders  = sum(product_time == 1),
    prod_second_orders = sum(product_time == 2)
  )
prd_tmp3 = head( prd, n = 40 )

prod_reorder_probability - probability of first re-order prod_reorder_times - aka prd$prod_orders / prd$prod_first_orders as prd$prod_first_orders + prd$prod_reorders = aka rd$prod_orders see prd_tmp3 : inverse of share of prod_first_orders in prod_orders, if that share is close to one, then it is very likely that this product on average was bought only once prod_reorder_ratio - share of reorders in orders

  • up_order_rate: user-specific order intensity of a product_id : proportion of times a product_id was ordered in total number of orders
